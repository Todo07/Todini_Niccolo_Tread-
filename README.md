

## descrizione
#### progetto in java che simula la produzionde di palette da giardinaggio, con due thread in parallelo:
-ProgettazioneManico - AssemblaggioLama​


## struttura
#### la struttura del progetto è formata da tre classi

## funzionamento
#### abbiamo tre classi dove la classe padre incomincia la produzione poi le altre due classi con i thread in parallelo finisco le varie fasi e la produzione stessa.
#### abbiamo usato join per aspettare il completamento nella classe padre e sleep sulle classi figlie per il tempo di produzione
